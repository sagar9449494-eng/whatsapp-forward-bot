# WhatsApp Forward Bot

This project automatically forwards messages from one WhatsApp number to another using Node.js and whatsapp-web.js.

## Numbers Used

Source Number: +91 70063 46953
Target Number: +91 96604 77174

## Installation

Install dependencies:

npm install

## Run the bot

npm start

## How it works

When a message is received from the source number, the bot automatically forwards it to the target number.
